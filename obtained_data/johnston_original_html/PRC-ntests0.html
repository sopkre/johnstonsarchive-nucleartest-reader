Database of nuclear tests, China-PRC: introduction

compiled by Wm. Robert Johnston
last modified 4 April 2009

  • Nuclear tests
Column data:
  • ID number--sequential number in listing
  • Shot name--Chinese name
  • Shot code--
    • S--salvo test, first listed detonation
    • SS--salvo test, additional detonations
    • X--not included in some listings
  • Year, month, day (GMT)
  • Time (GMT)
  • Site--
    • LN--Lop Nor
      • LN-A--Lop Nor, area "A"
      • LN-B--Lop Nor, area "B"
      • LN-C--Lop Nor, area "C"
      • LN-D--Lop Nor, area "D"
  • Latitude (° N) and Longitude (° E)
  • Location code
  • Hole designation--shaft or tunnel number
  • Height of burst--negative indicates depth
  • Ground zero altitude
  • Ground zero code
  • Type of detonation--
    • A--atmospheric
      • A-AD--airdrop
      • A-R--rocket or missile
    • AS--atmospheric, surface
      • AS-T--tower
    • UG--underground
      • UG-S--shaft
      • UG-T-tunnel
  • Purpose--
    • WR--nuclear weapons related
  • Yield, minimum and maximum--in kt, Western figures
  • Yield, estimated--in kt, reported figures from unofficial or Western sources
  • Yield notes
  • Yield, preferred value
  • Magnitude Mb--seismic
  • Crater--diameter in meters
  • Vented--radioactivity release in curies
  • Device
    • ER--enhanced radiation
    • FZ--fizzile
    • IC--implosion, composite core
    • IP--implosion, plutonium core
    • IU--implosion, uranium core
    • TN--thermonuclear, multistage (number indicates % fission)
  • Warhead
  • Sponsor
  • Reported
  • Notes
  • Sources--see list below
